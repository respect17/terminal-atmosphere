### Testing Instructions

To test Terminal Atmosphere:

1. **Installation**:
   ```bash
   cd terminal-atmosphere
   npm install
   ```

2. **Basic Commands**:
   ```bash
   node bin/atmosphere.js weather
   node bin/atmosphere.js analyze --depth basic
   node bin/atmosphere.js optimize --focus memory
   ```

3. **Real-time Monitoring**:
   ```bash
   node bin/atmosphere.js monitor --interval 5
   ```

4. **Profile Creation**:
   ```bash
   node bin/atmosphere.js profile --create demo-profile
   ```
